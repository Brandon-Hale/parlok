# parlok

Guardrails for agents. Stop bad tool calls before they happen.

A lightweight firewall for agent tools — every call is evaluated against declarative policies and routed to one of four outcomes: Allow, Rewrite, Approve, or Deny.

**Status:** v0 skeleton. Public shape defined; no real logic yet. Join the waitlist on the landing page.

## Layout

- `sdk/python/` — the Python SDK
- `web/` — the landing page (Next.js)
